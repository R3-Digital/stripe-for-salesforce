# Virtual Terminal

## What is a virtual terminal?

A virtual terminal is an online application that allows users to input and process card payments using an online application, bypassing the need for physical card-reading hardware.&#x20;

## Using Salesforce as a virtual terminal

The Stripe for Salesforce app is designed to empower users to leverage the payment processing capabilities of Stripe, transforming Salesforce into a virtual terminal for processing payments.&#x20;

The Stripe for Salesforce integration enables users to collect payment information from customers through the telephone, through emailed invoices, and/or payment links.
